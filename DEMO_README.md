## Important Features & Safety Notes

### ✓ Production-Safe Features
- **Accessibility Controls** — Fixed position (top-right), non-blocking design
- **Voice Features** — Text-to-speech only, with graceful browser fallback
- **Data** — All data is mock/sample, no real information
- **Integrations** — Demonstration interfaces only, no live connections
- **Security** — Demo credentials only (ARSLAN/Arslan@2030)

### ⚠️ Demo Limitations
- No real authentication backend
- No database persistence (session-only)
