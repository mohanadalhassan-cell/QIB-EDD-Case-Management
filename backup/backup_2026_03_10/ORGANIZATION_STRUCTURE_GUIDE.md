## 5.2 Image Handling & Fallbacks

### Smart Fallback System:

```javascript
// When photo fails to load:
<img src="assets/employees/NAME.jpeg" 
     alt="EMPLOYEE" 
     style="width:100%;height:100%;object-fit:cover;" 
     onerror="this.style.display='none';
              this.parentElement.innerHTML='
              <div style=\'width:100%;height:100%;
                    display:flex;align-items:center;
                    justify-content:center;
                    background:linear-gradient(135deg,COLOR1,COLOR2);
                    font-size:SIZE px;font-weight:800;color:#fff;\'>
              INITIALS</div>'">
```

### Features:
1. **Automatic Initials** - John Doe → JD
2. **Color Matching** - Gold for board, Color for department
3. **Fallback Design** - Professional gradient background
4. **Z-index Management** - Ensures proper layering
5. **CSS Animations** - Smooth transitions on hover

### Photo Dimensions:

| Chairman | 160px circular | Hero image |
| Board Members | 72px circular | Member cards |
| Exec Officers | 80px rounded | C-Suite cards |

# SECTION 10: MAINTENANCE & UPDATES

## 10.1 How to Update Organization Structure

### Adding New Employee:

```html
<div class="team-member" onclick="showDetail('NEW-001')">
  <div class="team-member-photo">
    <img src="assets/employees/NAME.jpeg" 
         alt="Full Name" 
         onerror="this.parentElement.innerHTML='<div...>XX</div>'">
  </div>
  <div class="team-member-info">
    <h5>Full Name</h5>
    <p>Job Title</p>
  </div>
</div>
```

### Updating Board Member Photo:

1. Save high-quality photo to QIB website or local assets
2. Update `src` URL in HTML
3. Fallback initials automatically generated
4. No reload needed - image loads dynamically

### Changing Color Scheme:

```css
--gold: #D4AF37;  /* Change board color */
--gold-dark: #9A7B0A;
--executive-blue: #1E40AF;  /* Change exec color */
```
