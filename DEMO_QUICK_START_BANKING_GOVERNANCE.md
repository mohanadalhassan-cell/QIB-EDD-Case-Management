```powershell
# Check Node.js version
node --version

# Check if port is available
Get-NetTCPConnection -State Listen | Where-Object {$_.LocalPort -eq 8585}
```

## 🆘 Quick Troubleshooting

| Issue | Solution |
|-------|----------|
| **Port 8585 in use** | `Get-Process node \| Stop-Process -Force` |
| **Can't connect** | Verify server is running, try `127.0.0.1:8585` |
| **Page looks broken** | Hard refresh: `Ctrl+Shift+Delete` |
| **Wrong login credentials** | Use exactly: **ARSLAN** / **Arslan@2030** |
| **JavaScript errors** | Check browser console (F12), refresh page |
