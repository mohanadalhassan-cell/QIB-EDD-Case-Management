## 🛠️ المهام الشائعة

### مهمة 1: تسجيل حالة جديدة

```
1. تسجيل الدخول
2. اذهب إلى Dashboard
3. اضغط "New Case"
4. أملأ البيانات
5. اختر المخاطر (LOW/MEDIUM/HIGH)
6. أضف المستندات
7. أرسل للمراجعة

⏱️ الوقت: ~5 دقائق
```

### مهمة 2: مراجعة حالة

```
1. تسجيل الدخول كـ Compliance Officer
2. اذهب إلى "Cases Under Review"
3. اختر حالة
4. راجع المستندات
5. قرّر (Approve/Reject/Request Info)
6. أضف تعليقات
7. احفظ

⏱️ الوقت: ~10 دقائق لكل حالة
```

### مهمة 3: البحث عن حالة معينة

```
1. افتح Dashboard
2. استخدم "Search" في الأعلى
3. اكتب اسم العميل أو ID
4. النتائج ستظهر فوراً
5. اضغط على الحالة لفتح التفاصيل

⏱️ الوقت: ~30 ثانية
```

### مهمة 4: تصدير تقرير

```
1. اذهب إلى Dashboard
2. اضغط "Export"
3. اختر النطاق الزمني
4. اختر الصيغة (CSV/PDF/Excel)
5. اضغط "Download"

⏱️ الوقت: ~1 دقيقة
```

## 🔒 نصائح الأمان

### 1. كلمات المرور

```
❌ لا تستخدم كلمة المرور مع شخص آخر
❌ لا تكتب كلمة المرور في ملف
✅ غيّر الكلمة بعد أول دخول
✅ استخدم كلمة قوية (12+ حرف)
```

### 2. البيانات الحساسة

```
❌ لا تنسخ أرقام ID الشخصية
❌ لا تشارك الـ JWT tokens
✅ استخدم HTTPS فقط (تلقائي على Netlify)
✅ سجّل الخروج عند الانتهاء
```

### 3. الجلسات

```
⏱️ تنتهي الجلسة تلقائياً بعد: ساعة واحدة
🔑 سجّل الدخول مجدداً للمتابعة
📍 الجلسة خاصة بـ Browser فقط
```

## ❓ أسئلة شائعة

### س: هل يمكن حذف بيانات تجريبية؟

**ج:** نعم، لكن تذكر:
- البيانات ستُعاد على تحديث الصفحة
- استخدم نسخة الإنتاج للبيانات الفعلية
- اطلب ملف نسخة احتياطية قبل الحذف

### س: كيف أُعيّن حالة لشخص آخر؟

**ج:**
1. أفتح الحالة
2. اضغط "Assign To"
3. اختر المستخدم
4. أضف ملاحظة
5. احفظ

### س: هل يمكن استعادة حالة محذوفة؟

**ج:** نعم! استخدم:
- سجل التدقيق (Audit Log) لرؤية الحذف
- النسخة الاحتياطية (Backup)
- اتصل بالمسؤول

### س: ما أقصى عدد مستخدم؟

**ج:** النظام يدعم:
- في الإصدار الحالي: 1000+ مستخدم
- مع قاعدة بيانات: غير محدود

### س: متى تُحدّث البيانات تلقائياً؟

**ج:**
- كل 30 ثانية يتم الحفظ
- مع كل change يتم التسجيل
- غير محتاج لضغط Save يدويّ
