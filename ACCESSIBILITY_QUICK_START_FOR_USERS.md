## Finding the Accessibility Button

### Where Is It?

Look in the **bottom-right corner** of your screen. You'll see a blue circular button with ♿ symbol.

```
┌─────────────────────────┐
│                         │
│     Main Content        │
│                         │
│              ♿ ← Here!  │
└─────────────────────────┘
```

### How to Open It

**Method 1:** Click the ♿ button  
**Method 2:** Press `Ctrl + Alt + A` on keyboard

---

## Feature 1: Adjust Text Size

### For People with Low Vision

**Make Text Bigger:**
1. Click ♿ button
2. Click "A+" button (or drag slider right)
3. Text gets larger
4. Click multiple times for even larger text

**Sizes Available:** 75% to 200%

### Example
```
75% = Small text (compact view)
100% = Normal (default)
150% = Large text
200% = Extra large text
```

---

## Feature 2: High Contrast Mode

### For People with Color Blindness or Low Vision

**Enable High Contrast:**
1. Click ♿ button
2. Click "Enable High Contrast"
3. Page turns black/white/yellow
4. Text becomes much clearer

**What Changes:**
```
BEFORE:     AFTER:
Dark blue → Black
White text → White text
Gray → Yellow (for accents)
```

**Disable Anytime:** Click "Enable High Contrast" again to turn off

---

## Feature 3: Voice Navigation

### For Hands-Free Operation

**Start Listening:**
1. Click ♿ button
2. Click "🎤 Start Voice" button
3. System says "Listening..."
4. Say your command clearly

**Available Commands:**
```
"next section"      → Go to next part
"previous section"  → Go back
"read page"        → Read content aloud
"submit"           → Submit form
```

### Example Conversation
```
You: "next section"
System: "Navigating to next section..."
✓ Page moves to Section 2
```

## Troubleshooting

### Voice Recognition Not Working?

**Check:**
- [ ] You said command clearly
- [ ] Microphone is unmuted
- [ ] You have internet connection
- [ ] Browser is Chrome or Edge
- [ ] Tab has focus (click page first)

**Try Again:**
1. Click ♿ button
2. Click "🎤 Start Voice" again
3. Wait for "Listening..."
4. Speak slowly and clearly

### Text Size Not Changing?

**Check:**
- [ ] You clicked "A+" or "A−" button
- [ ] Slider moved
- [ ] Wait 1 second for effect

**Reset:**
1. Click ♿ button
2. Click "Reset Defaults"
3. Then try again

### High Contrast Not Working?

**Check:**
- [ ] Click "Enable High Contrast" button
- [ ] Page should turn black/white
- [ ] Verify it says "Enable High Contrast" or "Disable..."

**Reset:**
1. Click "Reset Defaults" button
2. Try enabling again

### Audio Not Playing?

**Check:**
- [ ] Speakers/headphones connected
- [ ] Volume not muted (🔊 visible)
- [ ] Audio button shows "🔊 On"
- [ ] System volume not muted

## Common Questions

### Q: Will changing text size affect my investigation data?
**A:** No. Text size is only how you see information. All data remains the same.

### Q: Do other investigators see my accessibility settings?
**A:** No. Settings saved only on your device. Others see normal view.

### Q: Can I use voice commands in Arabic?
**A:** Currently English only. Arabic support planned for future version.

### Q: What if I don't see the accessibility button?
**A:** 
- Scroll to bottom-right corner
- It should be visible on all pages
- Try pressing Ctrl+Alt+A
- Refresh page

### Q: Can I customize the colors more?
**A:** Currently high contrast only. Custom themes coming soon.
