## 😊 IF AUDIENCE SEEMS CONFUSED

**Red flags:**
- No eye contact (lost them)
- Checking phones (not engaged)
- Blank stares (too fast)
- No note-taking (not important)

**Fix:**
1. **Stop mid-thought**
2. **Ask directly:** "Following me? Any questions so far?"
3. **Simplify:** Drop tech details, focus on value (40%, quality, compliance)
4. **Ask them:** "What's your biggest concern with manual EDD? Let me show you how we solve it."
5. **Re-engage:** Make it conversational, not lecture
